---
title: Quick Start
---

Get Cocommand running in minutes.

## Prerequisites

- macOS
- A GitHub account to download the DMG from releases
- (Optional) An OpenAI-compatible API key if you plan to use the built-in LLM

## Install

1. Download the latest Cocommand DMG from the GitHub Releases page.
2. Open the DMG and drag Cocommand into your Applications folder.
3. Launch Cocommand and grant any required permissions.

## First run

- Press the global shortcut to open the command bar.
- Type `/settings` to open the Settings window.
- If you want LLM responses, set your API key in the environment before launching.
