---
title: Built-in Apps
---

Built-ins are first-party Applications that ship with Cocommand.
Each built-in defines its capabilities, tool schemas, and UI behavior.

## Planned Built-ins (v0)

- Clipboard
- Calculator
- Notes

Each built-in page should include:
- Capabilities list
- Tool schemas (input/output)
- Permission scopes and risk levels
- UI behavior
- Examples and acceptance criteria
