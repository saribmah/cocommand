---
title: Milestones
---

Milestones define scope and order of delivery. Each milestone has explicit non-goals.

## v0 (Foundations)

- Command bar UI
- Routing + planning
- Permission layer
- Workspace snapshot + event stream
- Built-in apps: Clipboard, Calculator, Notes
- Extension host (Deno) with manifest + tool execution
- Observability: tool logs and recent actions

Non-goals:
- Extension marketplace
- Persistent chat sessions
- Cloud sync

## v1 (Ecosystem)

- Extension marketplace and signing
- Advanced routing personalization
- Expanded built-ins
- Undo for more tool classes
- Richer UI panels
